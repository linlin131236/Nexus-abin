# 🧠 Nexus

> Author：阿宾 | GitHub：[linlin131236/Nexus-abin](https://github.com/linlin131236/Nexus-abin)

## 3 步安装

### 1️⃣ 克隆 Skill

```bash
git clone https://github.com/linlin131236/Nexus-abin.git ~/.claude/skills/nexus
```

### 2️⃣ 创建知识库 + 连 Obsidian

```bash
mkdir -p ~/ai-brain-vault/{00_Inbox,01_Daily,02_Notes,03_Insights,04_Projects,05_Person,06_System}
```

然后打开 Obsidian → **"打开其他仓库"** → 选 `~/ai-brain-vault` 这个文件夹。**完成。** AI 写进去的笔记，Obsidian 实时看到。

### 3️⃣ 开始用

打开 Claude Code，AI 自动加载 SKILL.md，直接对话：

```
外脑 AI Agent 的最新进展是…
搜索外脑 Agent 框架
提醒 15:30 开会
```

---

> 💡 **原理**：SKILL.md 里路径写的是 `~/ai-brain-vault/`，Obsidian 打开同一个文件夹。Claude Code 写笔记 → Obsidian 实时可见。不是同步，是**同一个文件夹**。

### 也支持 Codex CLI

```bash
cp SKILL.md ~/.codex/skills/ai-brain/
```

## 怎么用

把 SKILL.md 当 AI 的系统提示词加载后，AI 自动识别以下触发词：

| 类别 | 触发词示例 | 做什么 |
|------|-----------|--------|
| 🧠 知识库 | `外脑 xxx`、`搜索外脑`、`外脑统计数据` | 写笔记/搜索/统计 |
| 🤖 AI 能力 | `总结链接`、`翻译`、`写文章`、`脑图` | AI 写作/翻译 |
| 📱 自媒体 | `选题灵感`、`标题优化`、`热搜` | 内容创作 |
| 🖥️ 电脑 | `截图`、`音量`、`锁屏`、`关机` | 电脑控制 |
| 🎮 生活 | `天气`、`提醒`、`番茄钟`、`选哪个` | 生活工具 |

详细触发词表见 [SKILL.md](SKILL.md)。

## 全部功能

### 🧠 知识库管理
| 触发词 | 做什么 |
|--------|--------|
| `外脑 <内容>` | 自动分类 → 深度加工 → 写笔记（Frontmatter + 标签 + 双向链接） |
| `搜索外脑 <关键词>` | ripgrep 全文搜索 vault，中文精准命中，返回 Top5 |
| `外脑待办` | 扫描所有 `- [ ]` 未完成任务 |
| `外脑问我` | 随机读 5 篇笔记，向用户提 3 个问题 |
| `外脑统计` | 笔记总数 / 分类分布 / 标签云 |
| `外脑重建` | 分析一周笔记 → 认知变化报告 |
| `外脑备份` | git commit + push |
| `外脑 年度报告` | 全年统计 + 高频词 + 月度趋势 |
| `外脑 今日回顾` | 查询去年今日的笔记 |
| `外脑 想法碰撞` | 随机两篇笔记 → AI 找隐藏关联 |
| `生成skill <需求>` | Fable5 六层框架生成专业 Skill 提示词 |

## 🧬 迭代记忆（核心特性）

装了这 Skill，AI 不再是"每次对话重新认识你"。

| 机制 | 做什么 |
|------|--------|
| **会话启动协议** | 每次开对话 → 检查提醒 + 今天写日记没 + 读用户档案 |
| **写入即进化** | 写新笔记 → 自动搜索旧笔记 → 补双向链接 → 更新用户档案 |
| **每周认知重建** | 周日自动扫描一周笔记 → 分析认知变化 → 问你 3 个问题 |
| **主动反问** | 每 10 篇笔记 → 随机挑 5 篇 → 提 3 个问题深化思考 |
| **用户档案自动维护** | `05_Person/user-profile.md` 全程 AI 更新：身份 / 偏好 / 高频话题 / 近期关注 |

## ✍️ 写作铁律 + 去 AI 味

所有 AI 生成内容自动遵守：

- 破折号数 = 0
- 禁用 "不是 X 而是 X" 句式
- 禁用 "竟然 / 仿佛 / 宛如"
- 双向链接 ≥ 3 个
- 不文艺腔不排比
- AI 书面语自动替换（"进行"→"做"、"因此"→"所以"、"综上所述"→"说人话就是"…）

## 🌙 Nexus Evolve — 自动Skill进化引擎

> 阿宾独立开发的每日学习引擎。Nexus 不只是工具——**越用越聪明**。

| 机制 | 做什么 |
|------|--------|
| **收割** | 每晚自动扫描 Claude Code / Codex 历史会话 |
| **挖掘** | AI 提取反复出现的任务、偏好、习惯 |
| **门控** | 只有 ≥3 个会话验证 + 比现有规则更优才通过 |
| **固化** | 通过门控的改进自动写回 CLAUDE.md |

```bash
python scripts/nexus-evolve.py run --auto-adopt  # 全自动
python scripts/nexus-evolve.py dry-run            # 仅看报告
python scripts/nexus-evolve.py status             # 查看最新
```

每天生成 `05_Person/Reflect-YYYY-MM-DD.md` 报告。
