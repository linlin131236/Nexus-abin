## 加载机制

此文件在项目根目录存在时，Claude Code 自动加载 SKILL.md 作为系统提示词。

## 加载后的行为

1. **会话启动**：检查提醒队列 → 检查今日日记 → 读取用户档案
2. **触发词拦截**：所有带「外脑」前缀的消息走写入管道
3. **管道不可绕过**：不得用 Write 工具直接写入 vault，必须走 SKILL.md 定义的流程
4. **写作铁律**：所有生成内容遵守破折号=0、无"不是X而是X"、双向链接≥3
5. **失败兜底**：工具不可用时告知用户，不自作主张绕过去

## 禁止

- 不得在非 vault 路径写入笔记
- 不得在未确认时执行关机/删除类操作
- 不得生成包含用户真实数据的内容给第三方

> 本文件是 SKILL.md 的加载器和安全守门人，不是装饰。
